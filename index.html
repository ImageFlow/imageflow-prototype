WordPress Image Flow

Feature Plugin Prototype

We have created scenarios involving using images in WordPress. They range from adding a new image to a post to creating a gallery. This prototype is the tool we have created to get user feedback and to prove out the new UX/UI ideas we have about image and media use within WordPress.

The Prototype
1. Add a New Image to a Post
2. Add an Existing Image to a Post
3. Edit and Add an Existing Image to a Post
3a. Edit Metadata and Add Existing Image to a Post
4. Create A Gallery and Insert into Post
5. Insert Multiple Items into Post
6. Edit Image that has been Inserted
